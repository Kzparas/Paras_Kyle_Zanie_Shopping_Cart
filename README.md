Features that i added
Product menu = View all products and each product shows name, category, price, and remaining stock

Cart management = Add items to cart, remove items to cart, update item quantity, and clear entire cart
Checkout system = Displays receipt with receipt number, date and time, purchased items, grand total, discount (10% if total ≥ 5000 ) final total. And for the payment validation it must be numeric number, must be enough to cover total. Display change
Stock management = Prevents buying more than available stock. Automatically deducts stock after adding to cart
Low stock alert = Displays product with stock ≤ 5 after check out
Order history = Stores completed transaction and display previous orders during program run
Input validation = Handles invalid inputs, ensures correct menu selection and validates Y/N responses
